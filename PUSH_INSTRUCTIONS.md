# Push Instructions

## Command line

### Using SSH
```bash
git remote add origin git@github.com:<YOUR_USERNAME>/eufm-xylella.git
git push -u origin main
```

### Using HTTPS
```bash
git remote add origin https://github.com/<YOUR_USERNAME>/eufm-xylella.git
git push -u origin main
```
